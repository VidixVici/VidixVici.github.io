VxV GenPass

Description

A simple password manager that is completely offline developed in Python. It utilizes the TKinter, Pyperclip, and Os libraries in Python as well to help assist some of the features in the program.

Features

- Stores unlimited Logins
- Randomly generates a strong password of any character length
- Can copy information to the Clipboard
